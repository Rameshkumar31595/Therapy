Narration MP3s for the Therapy Voice Guide live here:

  audio/en/<therapy>.mp3   (English)
  audio/te/<therapy>.mp3   (Telugu)

Therapies: padaabhyanga, abhyanga45, abhyanga60, potli, shirodhara, pizhichil

Generate all files from the project root:

  $env:ELEVENLABS_API_KEY = "sk_your_key_here"
  node generate-audio.js

See narration/README.md for voice selection and settings.
